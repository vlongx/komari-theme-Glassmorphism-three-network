# Komari Glassmorphism v2.3.0：高密度 mini 卡片、离线置底和更完整的主题设置说明

Komari Glassmorphism 是一款给 **Komari Monitor** 使用的毛玻璃风格主题。这次 v2.3.0 主要补上了高密度节点展示、离线节点排序优化，也重新整理了 README，把很多之前藏在主题设置里的功能和亮点写出来，方便新用户知道它到底能调什么。

## 这次更新重点

### 1. 新增 mini 节点卡片尺寸

如果你的节点很多，原来的卡片一屏可能放不下太多。v2.3.0 新增了 `mini` 尺寸：

- 卡片更矮、更紧凑；
- CPU / 内存 / 流量做了高密度布局；
- 延迟和丢包面板也同步压缩；
- 原有 `compact`、`comfortable`、`large` 都保留，不会影响已有布局。

可以在 Komari 后台主题设置里选择：

```text
节点卡片尺寸：mini / compact / comfortable / large
```

### 2. 新增「离线节点置底」

很多面板节点一多，离线机器混在前面会影响查看在线机器状态。这次新增了离线置底开关：

- 默认排序、月成本、总流量、上行、下行、峰值、即将到期等模式下，在线节点会优先显示；
- 离线节点排到最后；
- 「离线」快捷筛选仍然只显示离线节点，不会被这个设置影响。

### 3. 登录后可查看 hidden 节点

未登录公开首页仍然过滤 Komari hidden 节点；登录后则可以看到 hidden 节点，方便管理员自己检查隐藏机器状态。

### 4. 加载阶段显示更稳定

优化了顶部后台入口和访客信息组件的显示时机，避免页面刚加载时因为默认状态导致入口或访客 IP 组件闪一下。

### 5. 版本号改成单一来源

以后发布版本只需要改：

```json
{
  "version": "2.3.0"
}
```

也就是 `komari-theme.json` 顶层的 `version` 字段。构建时页面底部显示的版本号会自动读取这里，不需要再同时改多个文件。

## 很多人可能不知道的功能

这次 README 也重构了，把主题已有功能完整写出来。这里简单列一下：

### 首页总览卡片方案

主题内置多套头部卡片方案：

- 基础：常用内存、硬盘、剩余价值、流量、实时上下行；
- 运维：在线/离线、高负载、流量预警、连接峰值、平均 CPU/负载；
- 财务：即将到期、月费用、年费用、剩余价值；
- 流量：累计流量、实时峰值、上行最高、下行最高、流量预警；
- 完整：尽量展示全部统计项；
- 自定义：自己填写 key 控制显示内容和顺序。

### 快捷筛选 / 排序

首页分组旁边的快捷按钮支持：

- 默认；
- 月成本；
- 总流量；
- 上行 / 下行；
- 实时峰值；
- 离线；
- 高负载；
- 即将到期。

还可以设置进入首页默认使用哪个快捷模式。

### 三种地球 / 地图

主题支持三种首页地球/地图：

- realistic：真实贴图 3D 地球；
- cobe：点阵科技风地球；
- tiled：平铺世界地图，更适合节点多的场景。

也可以关闭地球旋转、隐藏地球，甚至隐藏整个头部模块，让首页变得更紧凑。

### 列表视图信息栏

除了卡片视图，列表视图也能显示更多节点信息：

- 厂商；
- 地区；
- 城市；
- ASN；
- 自定义标签；
- 分组。

这些字段可以自定义顺序，也可以关闭整行信息栏。

### 隐私相关设置

适合公开面板的几个设置：

- 未登录隐藏价格；
- 未登录隐藏后台入口；
- hidden 节点未登录不展示；
- 价格、剩余价值、费用类卡片可按登录状态隐藏。

### 自定义背景和公告

支持：

- 首页公告，内容可写简单 Markdown；
- 图片背景；
- 视频背景；
- 亮色 / 暗色分别设置背景；
- 背景模糊；
- 背景遮罩强度。

## 安装方式

1. 打开项目 Releases 页面；
2. 下载最新的 `komari-theme-Glassmorphism-build-*.zip`；
3. 进入 Komari 后台主题管理；
4. 上传 zip 主题包；
5. 在主题设置里按需调整卡片尺寸、地球样式、总览卡片、快捷控制和背景。

> 注意：请上传 Release 附件里的主题 zip，不要上传源码压缩包。
